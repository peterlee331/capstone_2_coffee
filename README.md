# ☕ Coffee Shop Sales Analysis & Forecasting

## Project Overview

This project analyzes coffee shop sales data to support business decision-making for 2026 budgeting.

The primary objective is to determine whether a **2–4% increase in sales** is achievable based on historical trends and predictive modeling.

Using a full data science pipeline—including data wrangling, exploratory data analysis (EDA), and machine learning—this project provides insights into key revenue drivers and actionable business recommendations.

This project answers:
> **Is this growth target realistic given current sales patterns?**

---

## Dataset

- Source: [Kaggle Coffee Sales Dataset]
- Time Period: March 2024 – March 2025  
- Key Variable:
  - `money` → Revenue generated per transaction
