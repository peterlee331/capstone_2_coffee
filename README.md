# ☕ Coffee Shop Sales Analysis & Forecasting

## 📌 Project Overview

This project analyzes coffee shop sales data to support business decision-making for 2026 budgeting.

The primary objective is to determine whether a **2–4% increase in sales** is achievable based on historical trends and predictive modeling.

Using a full data science pipeline—including data wrangling, exploratory data analysis (EDA), and machine learning—this project provides insights into key revenue drivers and actionable business recommendations.

---

## 🎯 Business Problem

The coffee shop aims to:
- Create a budget for 2026  
- Evaluate whether sales can increase by **2–4%**  

This project answers:
> **Is this growth target realistic given current sales patterns?**

---

## 📊 Dataset

- Source: [Kaggle Coffee Sales Dataset](https://www.kaggle.com/datasets/varishabatool/coffee-sales/data)
- Time Period: March 2024 – March 2025  
- Key Variable:
  - `money` → Revenue generated per transaction  

---

## 🔍 Key Insights (EDA)

- **Revenue Distribution:** Most transactions fall within a moderate range, with higher-value drinks contributing disproportionately to total revenue  
- **Product Impact:** Product type is the strongest driver of revenue  
- **Time Patterns:** Peak sales occur during morning to early afternoon  
- **Seasonality:** Monthly trends show fluctuations, indicating seasonal effects  

---

## 🤖 Modeling Approach

### Models Tested:
- Linear Regression  
- Ridge Regression  
- Random Forest  

### Final Model:
👉 **Linear Regression**
